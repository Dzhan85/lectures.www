-----------

Задания
-------

^^^^^^^^^

:l:`Pyramid`.

Задание 2, 3, 4
^^^^^^^^^^^^^^^

Делать не надо.

Содержание отчета
-----------------

Создать отчет, который должен быть оформлен в виде репозитария на :l:`GitHub`.

действий, результат выполнения заданий и выводы по работе.
